# WebTechnologies_Portfolio

A lightweight vanilla portfolio built with HTML, CSS and JavaScript.

## Project structure

- `index.html` — main homepage layout
- `style.css` — responsive light/dark styling
- `projects.js` — placeholder project objects and project data
- `main.js` — dynamic rendering, filters, theme toggle, and language switcher

## Notes

- No React, Next.js, Vue, Tailwind, npm, or build tools are used.
- Theme preference is saved in `localStorage`.
- Project cards are generated from `projects.js`.
- Category filters cover the five main creative collections.
